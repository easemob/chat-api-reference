muteExpireTimestamp property

int muteExpireTimestamp
final

获取当前被禁言截止时间戳（毫秒）。

加入聊天室后可获取。 当前用户被禁言或者被解除禁言时，此属性会被更新。

  • 当取值为0，表示当前用户未被禁言。
  • 当取值为-1，表示未能获取到用户被禁言时间戳。
